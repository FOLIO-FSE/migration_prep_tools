## Running

```bash
usage: generate_users.py [-h] [-l LOCALE] [-n NUM] [-d DOMAIN] [-u USER_GROUPS]

Generate Fake Data

optional arguments:
  -h, --help            show this help message and exit
  -l LOCALE, --locale LOCALE
                        Locale for provider. Default: 'en_US'
  -n NUM, --num NUM     Number of fake users. Default: 100
  -d DOMAIN, --domain DOMAIN
                        Fake domain for email addresses. Default: fake-ebsco.com
  -u USER_GROUPS, --user-groups USER_GROUPS
                        Text file of user groups.
```
